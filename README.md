<h1>AI Study Planner</h1>

<p>
  An AI-powered study planning application built using ReactJS, FastAPI, LangChain, and Groq LLM.
</p>

<h2>Overview</h2>

<p>
AI Study Planner is a full-stack AI productivity application that generates intelligent and personalized study schedules based on:
</p>

<ul>
  <li>Subjects</li>
  <li>Daily study hours</li>
  <li>Exam date</li>
</ul>

<p>
The application uses Large Language Models through LangChain and Groq to create structured study plans and visualize them using a modern productivity dashboard interface.
</p>


<h2>Features</h2>

<ul>
  <li>AI-generated personalized study plans</li>
  <li>LangChain-powered prompt workflow</li>
  <li>Groq LLM integration</li>
  <li>Modern ReactJS dashboard UI</li>
  <li>Weekly study planner</li>
  <li>Timeline-based study schedule visualization</li>
  <li>Learning insights panel</li>
  <li>Structured JSON output parsing</li>
  <li>FastAPI backend architecture</li>
  <li>Responsive UI design</li>
</ul>


<h2>Tech Stack</h2>

<h3>Frontend</h3>

<ul>
  <li>ReactJS</li>
  <li>CSS3</li>
  <li>Axios</li>
  <li>Vite</li>
</ul>

<h3>Backend</h3>

<ul>
  <li>FastAPI</li>
  <li>LangChain</li>
  <li>Groq API</li>
  <li>Pydantic</li>
  <li>Python</li>
</ul>


<h2>Application Workflow</h2>

<pre>
User Input
    ↓
React Frontend
    ↓
Axios API Request
    ↓
FastAPI Backend
    ↓
LangChain Prompt Processing
    ↓
Groq LLM
    ↓
Structured Study Plan
    ↓
Planner Dashboard UI
</pre>
